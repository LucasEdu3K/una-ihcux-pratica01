## 📌 Comandos mais utilizados no CMD Comecei 23/03/2026

- **dir**  
  Lista todos os arquivos e pastas do diretório atual.

- **mkdir [nome_da_pasta]**  
  Cria uma nova pasta/diretório.

- **cd ..**  
  Volta para o diretório anterior.

- **cls**  
  Limpa a tela do terminal.

- **cd [nome_da_pasta]**  
  Acessa uma pasta específica.

### Exemplo prático
```bash
mkdir projetos
cd projetos
dir
cls
